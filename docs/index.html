常驻菜单栏的 Claude Code 状态指示器

把 Claude Code 的工作状态变成一盏灯

Claude Traffic Light 通过 Claude Code hooks 监听会话事件，用红、黄、绿三种状态显示 AI 当前是否正在执行、已经完成，或需要你回到会话处理。

为高频 AI 编程工作流设计

它不是另一个聊天窗口，而是一个轻量、明确、始终可见的状态层。

01

实时状态指示

监听 Claude Code 关键 hooks，把 Start、Stop、AskUserQuestion 等事件映射为清晰灯色。

02

常驻菜单栏

窗口悬浮且不占任务栏，适合在 IDE、终端和浏览器之间频繁切换时保持可见。

03

单灯与三灯模式

经典三灯适合快速扫视，单灯模式适合更克制的桌面风格。

04

自动写入 hooks

首次启动会配置 Claude Code settings.json，并提供重新写入和手动配置入口。

05

主题与提示音

支持深色、浅色外观和可静音的状态提示音，减少长时间使用的干扰。

06

使用统计

记录红灯、绿灯次数和红灯持续时间，托盘菜单可查看本周使用概览。

三步启用

从安装到可用保持简单，核心配置由应用在本机完成。

下载应用

从 GitHub Releases 下载最新 macOS DMG 或 Windows 安装包。

启动红绿灯

应用会创建状态文件，并自动写入 Claude Code hooks 配置。

继续使用 Claude Code

正常提交任务即可，菜单栏会同步显示当前工作状态。

三种状态，足够明确

灯色只表达一件事：现在是否需要你关注 Claude Code。

绿灯

Claude 已完成回复，可以继续输入下一条指令或检查结果。

黄灯

Claude 正在思考、调用工具或输出内容，当前无需切回会话。

红灯

会话需要用户选择、输入或处理异常，应该回到 Claude Code 查看。

源码构建

需要从源码运行或自行打包时，可以使用下面这些命令。
