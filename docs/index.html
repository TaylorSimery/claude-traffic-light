特性Features

桌面安静,
关键时刻醒目。 Quiet on your desk.
Loud when it matters.

没有菜单栏图标,没有 Dock 图标,没有通知。只有一块小小的玻璃,告诉你 Agent 现在到哪一步了。 No menu bar clutter, no Dock icon, no notifications. Just one tiny pane of glass that tells you exactly where the agent is.

🌑

磨砂黑面板Matte-black panel

原生 SwiftUI 面板,深色磨砂质感与柔和投影,浮在所有 Space 之上。 Native SwiftUI panel with a deep matte finish and a soft drop-shadow. Floats above every Space.

⚡

实时读取会话Live session reading

每秒轮询 ~/.claude/projects/**/*.jsonl。无网络、无埋点、无守护进程。 Polls ~/.claude/projects/**/*.jsonl every second. No network, no telemetry, no daemon.

零占用Zero footprint

以 LSUIElement 后台运行。可执行文件约 250 KB。拖动移动,右键退出。 Runs as an LSUIElement background app. ~250 KB executable. Drag to move, right-click to quit.

🍎

纯 Apple 技术栈Pure Apple stack

SwiftUI + AppKit。没有 Electron、没有 Python、没有辅助脚本。Universal 二进制本地签名。 SwiftUI + AppKit. No Electron, no Python, no helper scripts. Universal binary signed for local run.

🔒

仅本地Local only

仅读取家目录中的文件,数据不离开本机。源码短小可审计。 Reads files in your home directory. Nothing leaves the machine. Source is short and auditable.

🛠

易于改造Hackable

状态识别逻辑全在一个 Swift 文件里 — 改规则、用 Xcode 重新编译、运行。 State logic lives in a single Swift file — adjust the rules, recompile in Xcode, run.

状态映射Status mapping

三种颜色,一个信号。 Three colors. One signal.

Claude Traffic Light 读取当前会话的最后一条消息和 CLI 进程状态,然后点亮其中一盏灯。 Claude Traffic Light reads the latest message in your active session and the CLI process state, then picks one light.

黄灯 — 思考中Yellow — Working

Claude 正在思考、流式输出或调用工具,会话日志在持续滚动。 Claude is thinking, streaming, or executing a tool. The session log is moving.

绿灯 — 已完成Green — Done

上一轮已经以 end_turn 干净结束,可以回来了。 The last turn ended cleanly with end_turn. Safe to come back.

红灯 — 需要你Red — Needs you

工具调用在等你确认、运行出错,或进程已退出。 A tool call is awaiting permission, the run errored, or the process exited.

安装Install

两种方式跑起来。 Two ways to get it running.

预编译应用 Prebuilt app

  1. 从 Releases 下载 ClaudeTrafficLight.zip。 Download ClaudeTrafficLight.zip from Releases.
  2. 把 ClaudeTrafficLight.app 拖到 /Applications。 Move ClaudeTrafficLight.app to /Applications.
  3. 首次启动:右键应用 → 打开,或运行下面命令去掉 quarantine 标记。 First launch: right-click the app → Open, or run the command below to drop the quarantine flag. 
# 清除 ad-hoc 签名构建的 quarantineclear quarantine for the ad-hoc signed build
xattr -dr com.apple.quarantine /Applications/ClaudeTrafficLight.app

从源码构建 Build from source

需要 Xcode 15+ 和 macOS 13+。 Requires Xcode 15+ and macOS 13+. 

git clone https://github.com/TaylorSimery/claude-traffic-light.git
cd claude-traffic-light
open ClaudeTrafficLight.xcodeproj   # 或者从命令行构建:or build from the CLI:
xcodebuild -project ClaudeTrafficLight.xcodeproj \
           -scheme ClaudeTrafficLight \
           -configuration Release \
           -derivedDataPath build clean build
