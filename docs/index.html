特性Features

需要时一眼看见，
不要时静悄悄。 Quiet on your desk.
Loud when it matters.

不进菜单栏，不上 Dock，也不弹通知。屏幕角落留一颗小灯，瞄一眼就够。 No menu bar clutter, no Dock icon, no notifications. Just one tiny pane of glass that tells you exactly where the agent is.

🌑

磨砂黑面板Matte-black panel

SwiftUI 原生绘制，磨砂质感配柔和投影。常驻所有桌面最上层，不被遮挡。 Native SwiftUI panel with a deep matte finish and a soft drop-shadow. Floats above every Space.

⚡

实时读取会话Live session reading

每秒扫一遍 ~/.claude/projects/**/*.jsonl。不联网、不上报、不跑守护进程。 Polls ~/.claude/projects/**/*.jsonl every second. No network, no telemetry, no daemon.

🪶

几乎不占资源Zero footprint

以 LSUIElement 后台运行，可执行文件约 250 KB。拖动改位置，右键即退出。 Runs as an LSUIElement background app. ~250 KB executable. Drag to move, right-click to quit.

🍎

纯 Apple 技术栈Pure Apple stack

SwiftUI + AppKit。没 Electron、没 Python、没外挂脚本。Universal 二进制，本地签名直接跑。 SwiftUI + AppKit. No Electron, no Python, no helper scripts. Universal binary signed for local run.

🔒

数据不出本机Local only

只读取本地用户目录下的日志，任何数据都不会离开这台电脑。代码简单，想自己审一遍也很快。 Reads files in your home directory. Nothing leaves the machine. Source is short and auditable.

🛠

想改随时改Hackable

状态判断都在一个 Swift 文件里：改规则、Xcode 重编、跑起来，三步搞定。 State logic lives in a single Swift file — adjust the rules, recompile in Xcode, run.

状态对照Status mapping

三种颜色，一目了然。 Three colors. One signal.

读取当前会话的最新消息，再结合 CLI 进程的状态，自动点亮对应的那一盏灯。 Claude Traffic Light reads the latest message in your active session and the CLI process state, then picks one light.

黄灯 · 正在干活Yellow — Working

Claude 正在思考、流式输出，或者在调用工具——会话日志一直在动。 Claude is thinking, streaming, or executing a tool. The session log is moving.

绿灯 · 这轮结束了Green — Done

上一轮已经干净结束，可以回来看结果了。 The last turn ended cleanly with end_turn. Safe to come back.

红灯 · 在等你Red — Needs you

工具调用在等你点确认，或者跑出了错，又或者进程已经退出。 A tool call is awaiting permission, the run errored, or the process exited.

安装Install

两种装法，挑一个顺手的。 Two ways to get it running.

直接装现成的 Prebuilt app

  1. 去 Releases 下载 ClaudeTrafficLight.zip。 Download ClaudeTrafficLight.zip from Releases.
  2. 把 ClaudeTrafficLight.app 拖进 /Applications 文件夹。 Move ClaudeTrafficLight.app to /Applications.
  3. 第一次打开：在应用上右键 → 打开；或者执行下面这条命令，去掉 macOS 的隔离标记。 First launch: right-click the app → Open, or run the command below to drop the quarantine flag. 
# 去掉 ad-hoc 签名应用的隔离标记clear quarantine for the ad-hoc signed build
xattr -dr com.apple.quarantine /Applications/ClaudeTrafficLight.app

自己从源码编译 Build from source

需要 Xcode 15 及以上、macOS 13 及以上。 Requires Xcode 15+ and macOS 13+. 

git clone https://github.com/TaylorSimery/claude-traffic-light.git
cd claude-traffic-light
open ClaudeTrafficLight.xcodeproj   # 或者直接走命令行：or build from the CLI:
xcodebuild -project ClaudeTrafficLight.xcodeproj \
           -scheme ClaudeTrafficLight \
           -configuration Release \
           -derivedDataPath build clean build
