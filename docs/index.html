Native macOS widget for Claude Code

把 Claude Code 的状态，变成桌面上的红绿灯。

不用反复切回终端。Claude Traffic Light 悬浮在所有窗口和全屏 Space 上，用红、黄、绿告诉你它是还在忙、已经结束，还是需要你回来处理。

像系统组件一样安静。

没有 Dock 图标，没有菜单栏图标，没有窗口按钮。它只在你需要的时候，用一个状态灯告诉你发生了什么。

原生 SwiftUI

没有 Electron、没有 Python、没有 App 内辅助脚本。下载后就是一个干净的 macOS App。

真正的桌面挂件

以 LSUIElement 后台应用运行，浮在所有窗口和 Space 之上，包括全屏应用。

本机读取日志

直接读取 `~/.claude/projects` 下的 Claude Code transcript，不上传数据，也不需要网络。

Claude Code 没运行、被中断、进程退出、工具报错，或者等你确认权限。

安装不会卡住。

不需要配置 hooks，不需要改 Claude Code 设置。

1

下载

从 GitHub Releases 下载 `ClaudeTrafficLight.zip`。

2

移动

解压后把 `ClaudeTrafficLight.app` 拖进 `/Applications`。

3

首次打开

右键 App 选择 Open。若 macOS 阻止打开，运行 `xattr -dr com.apple.quarantine /Applications/ClaudeTrafficLight.app`。

4

启动 Claude

在终端运行 `claude`，提交一次 prompt。信号灯会自动读取最新会话日志。

状态映射

claude-traffic-light
Claude Code process stopped       -> red
tool use / streaming / prompt     -> yellow
clean assistant end turn          -> green
permission / rejected / error     -> red

常见问题

怎么退出？

右键点击信号灯，选择退出。

一直红灯？

确认终端里正在运行 Claude Code，并且 `~/.claude/projects` 下出现了新的 `.jsonl` transcript。

能移动吗？

可以。按住挂件任意位置拖动。

有数据上传吗？

没有。App 只读取本机进程和 Claude Code 本地日志。
