实时显示 Claude Code 工作状态的优雅菜单栏应用

智能状态指示

🟢 绿灯

Claude 已完成回复
可以继续对话

🟡 黄灯

Claude 正在思考
输出或处理中

🔴 红灯

会话中断
需要用户输入或网络问题

快速开始

方式一：下载预编译版本

  1. 前往 Releases 页面
  2. 下载最新的 .dmg（macOS）或 .exe（Windows）
  3. 安装并启动应用

方式二：从源码构建

# 克隆仓库
git clone https://github.com/TaylorSimery/claude-traffic-light.git
cd claude-traffic-light

# 安装依赖
npm install

# 开发模式运行
npm run dev

# 打包 DMG（macOS）
npm run dist

# 打包 NSIS（Windows）
npm run dist:win

工作原理

应用通过 Claude Code 的 hooks 系统监听关键事件：

Start → 黄灯（Claude 开始输出）
Stop → 绿灯（Claude 完成回复）
AskUserQuestion → 红灯（需要用户选择）
UserPromptSubmit → 黄灯（用户提交消息）

首次启动时，应用会自动在 ~/.claude/settings.json 中配置必要的 hooks。

系统要求

macOS: 10.13 或更高版本

Windows: Windows 10 或更高版本

Claude Code: 需要安装 Claude Code CLI 或桌面版
